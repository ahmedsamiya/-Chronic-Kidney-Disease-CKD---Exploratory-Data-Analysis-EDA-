  Chronic Kidney Disease (CKD) - Exploratory Data Analysis (EDA)

##  Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the **Chronic Kidney Disease (CKD)** dataset to understand the data, identify patterns, detect missing values, analyze feature distributions, and prepare the dataset for Machine Learning.

The objective is to gain meaningful insights into the factors associated with Chronic Kidney Disease and build a strong foundation for predictive modeling.

---

## Objectives

- Understand the dataset structure
- Explore numerical and categorical features
- Handle missing values
- Detect duplicate records
- Analyze feature distributions
- Identify outliers
- Perform correlation analysis
- Visualize relationships between variables
- Prepare the dataset for Machine Learning

---

## 📂 Dataset

**Dataset Name:** Chronic Kidney Disease Dataset

The dataset contains various medical attributes such as:

- Potassium
- Hemoglobin
- Packed Cell Volume
- White Blood Cell Count
- Red Blood Cell Count
- Hypertension
- Diabetes Mellitus
- Coronary Artery Disease
- Appetite
- Pedal Edema
- Anemia
- Class (CKD / Not CKD)
